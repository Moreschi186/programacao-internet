PHP 5 Constantes

Constantes são como variáveis, exceto que, uma vez que são definidas, elas não podem ser alteradas ou indefinidas.

Uma constante é um identificador (nome) para um valor simples. O valor não pode ser alterado durante o script. Um nome de constante válido começa com uma letra ou um sublinhado (nenhum sinal de $ antes do nome constante). Nota: Ao contrário das variáveis, as constantes são automaticamente globais em todo o script.

Criar uma constante PHP

Para criar uma constante, use a função define().

Sintaxe

define(name, value, case-insensitive)

Parâmetros

  • name: Especifica o nome da constante
  • value: Especifica o valor da constante
  • case-insensitive: especifica se o nome constante deve ser sensível a maiúsculas e minúsculas

O exemplo abaixo cria uma constante com um nome sensível a maiúsculas e minúsculas:

		<?php
		define("GREETING", "Welcome to W3Schools.com!");
		echo GREETING;
		?>

O exemplo abaixo cria uma constante com um nome não sensível a maiúsculas e minúsculas:

		<?php
		define("GREETING", "Welcome to W3Schools.com!", true);
		echo greeting;
		?>

Constantes são globais

Constantes são automaticamente globais e podem ser usadas em todo o script.

O exemplo abaixo usa uma constante dentro de uma função, mesmo que seja definida fora da função:

		<?php
		define("GREETING", "Welcome to W3Schools.com!");
		function myTest() {
    	echo GREETING;
		}
		myTest();
		?>
